Cambridge A Level Computer Science (9618) asks for two different kinds of thinking, and examines them in two different ways. Half the syllabus is theory — how a processor fetches an instruction, how data moves across a network, how a database stays consistent. The other half is craft: writing code that runs. Our Computer Science Cambridge A Level online tuition is built around the fact that most students are comfortable with one and quietly avoid the other.

The structure is unusual and worth understanding before you start. There are four papers and **not one of them is multiple choice** — every mark is earned by writing something. Papers 1 and 3 are the theory papers, sat on paper. Paper 2 is where students get caught: it tests programming, but it is a *written* exam, and candidates do **not** write program code — they answer in pseudocode, using Cambridge's own built-in functions supplied on an insert. Paper 4 then flips it entirely: two and a half hours on a computer, no internet, writing real working code in Java, Visual Basic or Python (console mode), submitting the program plus evidence that it was tested. No evidence, no marks. Calculators are banned in all four papers.

Students who prepare for only one of those two skills lose a quarter of the qualification. So we teach them as separate disciplines from the start: pseudocode fluency and dry-running by hand for Paper 2, and real code written under time pressure for Paper 4. The theory follows the syllabus order that builds — information representation, hardware and processor fundamentals, then system software, security and databases at AS, opening into data representation, virtual machines, artificial intelligence and further programming at A Level.

Classes are live and capped small. Students write code while we watch, which is the only way to catch the habits that cost marks — the untested edge case, the loop that works on the example and nothing else, the pseudocode that would never actually run. One note for families: Paper 4 is sat on centre computers without internet access, so wherever your child is studying with us, they will need an exam centre that can host it. We will help you check that before you commit.
